# Игра "Червь"

[Deutsch](./README.md)

Это старая игра. Игрок управляет червём, который постоянно движется по полю и хочет есть. На поле появляется еда и игрок направляет червя к еде. После того, как червь что-то съест, он растёт. Если червь ударяется о края поля или кусает сам себя, то он погибает./

Цель игры - сохранять червя как можно должше живым.
