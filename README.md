# Das Spiel "Wurm"

[Русский](./README.ru.md)

Das ist ein altes Spiel. Ein/e Spieler/in steuert den Wurm, der ständig auf einem Feld kriecht und essen möchte. Auf dem Feld erscheint das Essen und ein/e Spieler/in führt den Wurm zum Essen. Nach jedem Mahlzeit wächst der Wurm. Wenn der Wurm geht gegen einen Rand des Felds oder beißt sich, dann kommt er um Leben./

Das Ziel des Spiels ist, den Wurm so lange wie möglich lebend aufzubewahren.
